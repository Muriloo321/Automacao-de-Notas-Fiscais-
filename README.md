# Automa-o-de-Notas-Fiscais-
Projeto de análise de dados simulados do SAP (ME80FN e KSB1) em Python. Realiza tratamento, integração e aplicação de regras de negócio para classificar pedidos por timing de lançamento (no mês, M+1, M+2+ e pendentes), gerando indicadores que apoiam análises operacionais.
